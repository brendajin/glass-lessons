Why Phonegap?

  • Think Interaction
  • Prototype quickly
  • Use familiar tools
  • Learn and test
I believe in ideas more than code. I believe in the human component of programming. I myself just learned to code January last year. I KNOW that Lots of people have great ideas. and I subscribe to the Arduino philosophy. Let's lower the barrier of entry so that people can test their them!

Grab Sensor Data

Create a basic UI


// Device APIs are available
document.addEventListener('deviceready',onDeviceReady);

// Callback function to execute when the device is ready
function onDeviceReady() {
    navigator.accelerometer.watchAcceleration(onAccelSuccess, onError, {frequency:1000});
}

// onAccelSuccess: Get a snapshot of the current acceleration
function onAccelSuccess(acceleration) {
    // Show results in document body
    document.getElementById('accel').innerHTML='Acceleration X: ' + acceleration.x + '\n' +
    'Acceleration Y: ' + acceleration.y + '\n' +
    'Acceleration Z: ' + acceleration.z + '\n' +
    'Timestamp: '      + acceleration.timestamp + '\n';
}
